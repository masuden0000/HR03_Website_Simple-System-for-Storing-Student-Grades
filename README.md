## Run Locally

Clone the project

```bash
  git clone https://github.com/HR05_Website_Simple-System-for-Storing-Student-Grades
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
